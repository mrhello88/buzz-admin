# SMM Panel Admin Dashboard

A professional admin dashboard built with Next.js, React, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

Dependencies are already installed. If you need to reinstall:

```bash
npm install --legacy-peer-deps
```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the dashboard.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Features

- **Dashboard Overview** - View statistics and recent activity
- **User Management** - Manage users with CRUD operations
- **Orders Management** - Track and manage orders
- **Payments** - View and manage payment records
- **Services** - Manage available services
- **API Integration** - Configure API integrations

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Radix UI Components
- Lucide React Icons
